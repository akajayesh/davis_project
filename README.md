This project analyzes restaurant data using Google Sheets pivot tables and charts to extract business insights.

## Dataset

Cleaned dataset contains restaurant details such as:

* Area
* Cost for two
* Rating
* Delivery time
* Cuisine type
* Restaurant type
* Outlet count

## Tools Used

* Google Sheets
* Pivot Tables
* Charts

## Key Insights

1. Dining prices vary significantly across locations.
2. Popular cuisines dominate customer demand.
3. Price weakly affects ratings.
4. Delivery time has minimal impact on satisfaction.
5. Focused menus outperform large menus.
6. Restaurants cluster in specific areas.
7. Some restaurant types rate higher than others.
8. Single outlets rate slightly higher than chains.
